This is the simple news application using retrofit + gnewsapi.io
🔑 API Key Setup

This project uses the NewsAPI service to fetch news. For security reasons, the API key is not included in this repository.

Step 1: Get an API Key
Visit [https://newsapi.org/](https://gnews.io/)
Create a free account.
Generate your API key.

step 2: replace api key 
In NewsRepositoryImpl.kt replace BuildConfig.API_KEY with your api key
